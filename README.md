# Ultimate Tic-Tac-Toe

A two-player Ultimate Tic-Tac-Toe Android app built with Java in Android Studio.

## What is Ultimate Tic-Tac-Toe?
A strategic variant of Tic-Tac-Toe played on a 3×3 grid of smaller Tic-Tac-Toe boards. 
Win 3 mini boards in a row to win the game. Where you play determines where your opponent must play next.

## Features
- Two player pass-and-play on same device
- Full send rule implementation
- Board highlighting for active board
- Win detection for mini boards and big board
- Match result screen with rating and reactions
- How to Play guide with section navigation

## Built With
- Java
- Android Studio
- Native Android UI components only
